# Wingspan Game Components

This document outlines all the components included in the base game of Wingspan.

| Component                | Quantity | Description                                                                                             |
| ------------------------ | -------- | ------------------------------------------------------------------------------------------------------- |
| **Bird Cards**           | 170      | Unique cards representing different bird species, each with specific attributes and powers.             |
| **Bonus Cards**          | 26       | Cards that provide players with secret end-of-game scoring objectives.                                    |
| **Player Mats**          | 5        | Double-sided mats representing each player's wildlife preserve, with three habitat rows.                |
| **Goal Tiles**           | 8        | Double-sided tiles that establish the public scoring goals for each of the four rounds.                 |
| **Food Tokens**          | 103      | Cardboard tokens representing five different food types: invertebrate, seed, fish, fruit, and rodent.    |
| **Egg Miniatures**       | 75       | Plastic egg miniatures in various colors, used to represent eggs laid by birds.                         |
| **Action Cubes**         | 40       | Wooden cubes in five different player colors (8 per player), used to select actions.                    |
| **Birdfeeder Dice Tower**| 1        | A cardboard dice tower used to roll the custom food dice.                                                 |
| **Custom Food Dice**     | 5        | Wooden dice with faces representing the different food types available in the game.                     |
| **Bird Tray**            | 1        | A plastic tray designed to hold the face-up bird cards available for players to draw.                   |
| **Goal Board**           | 1        | A board used to display the end-of-round goal tiles.                                                    |
| **Scorepad**             | 1        | A pad of sheets for scoring at the end of the game.                                                     |
| **First Player Token**   | 1        | A token used to mark the first player for the current round.                                            |
| **Rulebook**             | 1        | The main rulebook for the game.                                                                         |
| **Appendix**             | 1        | A supplementary booklet with details on bonus cards, end-of-round goals, and specific bird powers.      |
