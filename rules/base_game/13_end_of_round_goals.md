# End-of-Round Goals

This document provides a detailed reference for all end-of-round goals in Wingspan, as described in the appendix.

| Goal Name                               | Description                                                                                                                                                              |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Birds in Habitat**                    | Count the total number of birds you have in a specific habitat (Forest, Grassland, or Wetland). There is one copy of this goal for each habitat.                           |
| **Eggs in Habitat**                     | Count the total number of eggs you have laid on birds in a specific habitat. Multiple eggs on a single bird all count toward this goal.                                     |
| **Birds with Nest Type and Eggs**       | Count the total number of birds with a specific nest type (Platform, Bowl, Cavity, or Ground) that have at least one egg. Each bird counts only once, regardless of the number of eggs it has. Star nests are wild and count for any nest type. |
| **Eggs in Nest Type**                   | Count the total number of eggs on birds with a specific nest type. Multiple eggs on a single bird all count toward this goal. Star nests are wild and count for any nest type. |
| **Sets of Eggs in Different Habitats**  | Count the number of sets of eggs, where one set consists of at least one egg in the Forest, one in the Grassland, and one in the Wetland.                                   |
| **Total Birds**                         | Count the total number of birds you have played across all habitats.                                                                                                     |
