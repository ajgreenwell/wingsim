# Wingspan Glossary

This document provides definitions for key terms, icons, and concepts used in Wingspan.

## Game Terms

| Term | Definition |
|---|---|
| **Action Cube** | A wooden cube used to mark which action a player is taking on their turn. Players have 8 action cubes in their color. |
| **Birdfeeder** | The dice tower that holds the five custom food dice. Players gain food by selecting dice from the birdfeeder. |
| **Bird Tray** | The plastic tray that holds three face-up bird cards available for players to draw. |
| **Bonus Card** | A secret end-game scoring objective that players keep hidden until final scoring. |
| **Cache** | To place a food token on a bird card, where it remains for the rest of the game and is worth 1 point. |
| **End-of-Round Goal** | A public objective that players compete to achieve each round, earning points based on their performance. |
| **Habitat** | One of the three rows on a player mat: Forest (food), Grassland (eggs), or Wetland (cards). |
| **Tuck** | To place a card face-down underneath a bird card, where it remains for the rest of the game and is worth 1 point. |

## Food Types

| Icon | Food Type |
|---|---|
| Invertebrate | Represents insects, worms, and other invertebrates. |
| Seed | Represents seeds and grains. |
| Fish | Represents fish and aquatic creatures. |
| Fruit | Represents berries and other fruits. |
| Rodent | Represents mice, voles, and other small mammals. |
| Wild | Can be paid with any food type. |
| No Food | A crossed circle indicates the bird has no food cost. |

## Nest Types

| Icon | Nest Type |
|---|---|
| Platform | A flat nest built on a platform or branch. |
| Bowl | A cup-shaped nest. |
| Cavity | A nest in a hole or cavity. |
| Ground | A nest built on the ground. |
| Star | A wild nest type that matches any of the four nest types. |

## Bird Power Icons

| Icon | Meaning |
|---|---|
| Brown Background | "WHEN ACTIVATED" - Activates when you use that habitat's action. |
| Pink Background | "ONCE BETWEEN TURNS" - Activates on opponents' turns, once between your turns. |
| No Background | "WHEN PLAYED" - Activates only when the bird is played. |
| Predator Icon | Indicates the bird is a predator. |
| Flocking Icon | Indicates the bird has a flocking power (tuck cards). |

## Other Concepts

| Term | Definition |
|---|---|
| **Wingspan** | The measurement shown on bird cards, used for some bird abilities and bonus cards. |
| **Egg Limit** | The maximum number of eggs that can be placed on a bird, shown by egg icons on the card. |
| **Column Cost** | The egg cost to play a bird in a specific column: Column 1 (0 eggs), Columns 2-3 (1 egg), Columns 4-5 (2 eggs). |
